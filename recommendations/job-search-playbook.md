# Job Search Playbook

## Weekly Cadence

### Monday
- Review target companies
- Tailor resume for 2–3 strong-fit roles
- Submit priority applications

### Tuesday–Wednesday
- Reach out to contacts and alumni
- Send follow-ups for applications older than 5–7 days
- Prepare for interviews or screening calls

### Thursday
- Review rejections and patterns
- Improve one resume bullet, one project story, or one interview answer

### Friday
- Update all CSV trackers
- Run `python3 tools/job_tracker.py summary`
- Run `python3 tools/job_tracker.py recommend`
- Plan next week’s top 5 roles

## Practical Rules
- Prioritize quality over mass-applying
- Keep a list of 20–30 target companies
- Follow up politely and consistently
- Track referrals separately from cold applications
- Reuse strong stories, but tailor every submission
